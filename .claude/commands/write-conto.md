# /write-conto

Escreve um conto literário para um personagem de Korú.

## Como usar

```
/write-conto [personagem]
```

**Personagens disponíveis:** amara | oruku | beku | obaru | kemdi | temi | orike

**Exemplos:**
- `/write-conto amara`
- `/write-conto beku`
- `/write-conto orike`

## O que este comando faz

1. Lê `koru-ecosystem-briefing.md` para contexto do mundo
2. Lê `koru-workflow.md` para o perfil específico do personagem
3. Verifica se existe material do personagem no livro para não contradizer
4. Escreve o conto seguindo as especificações de tom e física de Korú
5. Salva em `contos/koru-conto-[personagem].md`

## Especificações

- **Tom:** narrativo literário. Melancolia funcional. Beleza que custa.
- **Extensão:** 1.500 a 3.000 palavras
- **Estrutura:** cabeçalho de referência (interno) + texto literário
- **Física:** integrada como dado, não explicada
- **Morfologia:** quadrúpedes com chifres. Zero mãos. Toque pela testa/focinho.

3. beku
